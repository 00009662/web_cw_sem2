# TO DO LIST

This app allows users to track the to dos.

#### Intructions 
1.Download the source code

```bash
git clone https://github.com/00009662/web_cw_sem2
```
or just download the *.zip file

2.Install deps
```bash
npm install
```

3.Run the app
```bash
node app
```

#### Deps list
- express
- pug
